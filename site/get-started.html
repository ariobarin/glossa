Glossa What it is

Installation

Connect a workspace in three steps.

Glossa is a Windows-first open beta. It connects ChatGPT to one Git repository that you choose from your computer.

1. Install the CLI

Install Git and Node.js 24, then run this command in PowerShell:

npm install --global @ariobarin/glossa@beta

2. Start Glossa

Change to the repository you want ChatGPT to use and start the worker:

Set-Location C:\path\to\your\project
glossa

Leave this terminal open while you work. Press Ctrl+C to disconnect.

3. Add it to ChatGPT

During the open beta, Glossa is added manually as a custom app in ChatGPT Developer Mode.

  1. Enable Developer Mode for your ChatGPT plan and workspace role.
  2. Open ChatGPT Settings, choose Apps, then choose Create.
  3. Name the app Glossa and enter https://mcp.glossa.sh/mcp.
  4. Choose OAuth authentication, scan the tools, and authorize the same Glossa account.
Verify the connection

Open a conversation with the Dev-labeled Glossa app selected and ask:

List my connected devices.

A connected worker reports one online device with root path ..

Expose only a workspace you trust.

Glossa can modify files and run commands with your Windows account's permissions. Do not expose your home directory, a drive root, or a repository containing credentials.
