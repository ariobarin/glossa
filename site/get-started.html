Get started

Connect a workspace in minutes.

Glossa is a Windows-first open beta. Use a disposable Git repository for your first connection.

1. Install Glossa

Install Node.js 24, then run this command in PowerShell:

npm install --global @ariobarin/glossa@beta

2. Start the worker

Open a disposable repository and start Glossa. Browser sign-in opens automatically when needed.

Set-Location C:\path\to\a\test-repo
glossa

Leave the terminal open while you use Glossa. Press Ctrl+C to disconnect.

3. Add Glossa to ChatGPT

Developer Mode is required during the open beta.

Glossa is not listed in the public plugin directory yet. Enable Developer Mode in ChatGPT, then install Glossa Live and complete authorization with the same account you used for the CLI.

  1. Follow OpenAI's Developer Mode guide for your plan and workspace role.
  2. Open Glossa Live and choose Install plugin.
  3. Complete authorization using the same Glossa account as the worker.

4. Verify the connection

Start a ChatGPT conversation with Glossa Live selected and ask:

List my connected devices.

Then try a harmless read from the disposable repository:

Open my connected Glossa workspace and read README.md.
Glossa can modify files and run commands.

Commands run with the full environment and permissions of your Windows account. Do not expose your home directory, a drive root, or a repository containing credentials.

For troubleshooting and more detail, read the open beta guide.
