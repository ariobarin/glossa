Glossa connects ChatGPT to one local coding workspace that you choose.

Start in 3 steps

Quickstart

From install to first prompt.

Windows and Node.js 24 are required.

  1. 1

    Connect a workspace

    Paste this into PowerShell. Sign in when your browser opens.

    npm install --global @ariobarin/glossa@beta
Set-Location C:\path\to\your\project
glossa

    Leave this terminal open. Glossa can modify files in this folder and run commands with your Windows account permissions. Press Ctrl+C to disconnect.

  2. 2

    Install the ChatGPT plugin

    Use the same account you chose when Glossa opened your browser.

    Install Glossa Live
  3. 3

    Send your first prompt

    Start a ChatGPT chat with Glossa Live enabled, then paste:

    Open my connected Glossa workspace and read README.md.

Start with a disposable Git repository. Never expose your home folder, a drive root, or a folder containing credentials.

Glossa is an execution bridge, not an agent. Security model
