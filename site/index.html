Connect ChatGPT to your computer.

Glossa connects ChatGPT to one local coding workspace that you choose.

Quickstart

Install once. Run glossa.

Glossa opens browser sign-in automatically and connects the current Git worktree. Windows and Node.js 24 are required. 

npm install --global @ariobarin/glossa@beta
Set-Location C:\path\to\your\project
glossa
Get started Read the full guide

Start with a disposable repository. Glossa can modify files in the exposed workspace and run commands with your Windows account permissions. Press Ctrl+C to disconnect.

Glossa is an execution bridge, not an agent.
