Quickstart

Install once. Run glossa.

Glossa opens browser sign-in automatically and connects the current Git worktree. Windows, Node.js 24, and eligible ChatGPT Developer Mode access are required. 

npm install --global @ariobarin/glossa@beta
Set-Location C:\path\to\your\project
