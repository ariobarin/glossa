Why Glossa?

ChatGPT can tell you what to change. Glossa lets it make the change.

Your files and tools live on your computer. ChatGPT on the web cannot reach them.

Glossa connects the chat to a folder you choose. It can read files, make changes, and run commands there. The connection lasts only while Glossa is running.

01 Your workspace stays local

The worker reads files and runs commands on your computer. The hosted relay does not clone your repository.

02 You choose the boundary

Start Glossa in one folder. That folder becomes the explicit root for file operations.

03 You control the connection

Glossa is available only while the local worker is running. Stop the process to disconnect it.
