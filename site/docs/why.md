# Why Glossa?

ChatGPT can tell you what to change. Glossa lets it make the change.

Your files and tools live on your computer. ChatGPT on the web cannot reach them.

Glossa connects the chat to a folder you choose. It can read files, make changes, and run commands there. The connection lasts only while Glossa is running.

<div class="why-points">
  <section>
    <span>01</span>
    <strong>Your workspace stays local</strong>
    <p>The worker reads files and runs commands on your computer. The hosted relay does not clone your repository.</p>
  </section>
  <section>
    <span>02</span>
    <strong>You choose the boundary</strong>
    <p>Start Glossa in one folder. That folder becomes the explicit root for file operations.</p>
  </section>
  <section>
    <span>03</span>
    <strong>You control the connection</strong>
    <p>Glossa is available only while the local worker is running. Stop the process to disconnect it.</p>
  </section>
</div>
