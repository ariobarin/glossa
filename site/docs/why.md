# Why Glossa?

ChatGPT can tell you what to change. Glossa lets it make the change.

Your files and tools live on your computer. ChatGPT on the web cannot reach them.

Glossa connects the chat to a folder you choose. It can read files, make changes, and run commands there. The connection lasts only while Glossa is running.
