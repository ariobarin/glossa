Put ChatGPT to work.

Glossa lets ChatGPT read files, make changes, and run commands in your folder.

ChatGPT.com Your folder

Install Glossa

Requires Node.js 24.

npm install --global @ariobarin/glossa@beta

Start it in a folder

Open a terminal in the folder where you want ChatGPT to work.

glossa .

Add it to ChatGPT

  1. Open Settings > Apps > Advanced Settings. Enable Developer Mode, then choose Create.
  2. Name it Glossa and paste this server URL:
https://mcp.glossa.sh/mcp
  1. Choose OAuth, check the confirmation box, then choose Create.
  2. Choose Sign in with Glossa.

Test the connection

In ChatGPT, send:

Use Glossa to list my connected devices.
