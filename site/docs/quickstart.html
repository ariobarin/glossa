Put ChatGPT to work.

Glossa lets ChatGPT read files, make changes, and run commands there.

ChatGPT.com Your folder

Install Glossa

Requires Node.js 24.

Terminal 
npm install --global @ariobarin/glossa@beta

Start it in a folder

Open a terminal in the folder where you want ChatGPT to work.

Terminal 
glossa .

Add it to ChatGPT

Open Settings > Apps > Advanced Settings. Enable Developer Mode, then choose Create.

  1. Name the app Glossa.
  2. Paste this MCP server URL:
MCP server URL 
https://mcp.glossa.sh/mcp
  1. Choose OAuth, then Scan Tools.
  2. Authorize Glossa, wait for the scan, then choose Create.
ChatGPT prompt 
Use Glossa to list my connected devices.
